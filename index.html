Hi, I'm Brittany. I'm a Seattle-based full stack developer who delights in creative problem solving, strong code, and out-of-the-box thinking. I am #blessed to be doing what I love.

I'm a full stack developer with a special place in my heart for front end work. Here are a few things I can't get enough of.

HTML5 & CSS3

I believe in a responsive foundation - this means strong HTML5 and CSS3. I take pride in knowing my work can be viewed and enjoyed on a variety of formats. It is truly amazing what all can be accomplished with good CSS3. I love playing with transitions, animations, and typography in my downtime.

jQuery

jQuery is an integral part of the modern web, and I love using it for anything ranging from animation to data manipulation. I have experience writing my own scripts as well as using popular plug-ins, like fullpageJS, parallax, and more.

AngularJS

I love how abstract AngularJS is. The beauty of a one-page application with two-way databinding in the MEAN stack inspires me so much. I have built several client-side applications with AngularJS, including my final project for the General Assembly Immersive. AngularJS is by far my favorite framework for client-side applications.

Node.js

Node.js is really powerful for building full stack applications since it's so easily combined with various frameworks. Given it's more of a traditional full stack set up, it gives you a lot more muscle for building database-reliant structures. I particularly enjoy working with Express, EJS and Sequelize with my Node.js apps. I've built a few applications with Node.js, but this is my favorite.

The possibilities of the web are endless. Here are a few of my favorite recent contributions.

Showup

A MEAN Stack App

Showup is a digital manifestation of the poster-covered street poles I walk past every day in Seattle. Using the AngularJS MEAN stack I created a beta version of this site for my final project at General Assembly with full CRUD and RESTful routing for visitors, users, and admin to update and see poster content. This is my pet project and I am actively adding to it.

Technologies used

  • AngularJS
  • Mongo
  • JWT
  • Heroku
  • HTML5
  • Cloudinary
  • CSS3
  • Bootstrap UI

RSVPtree

A Full-Stack App

RSVPtree is a Node app built with ExpressJS. It streamlines the RSVP process for big parties like weddings and family reunions. It uses Sequelize to track the database and form information of various parties and keep them separate while focusing on what information is needed from guests to efficiently plan a party. It's refreshingly simple to use as a guest. But the best part about it is how it forms family trees to help learn family connections better. This comes in handy if the user doesn't know their better half's extended family super well!

Technologies used

  • Express
  • Node.js
  • Sequelize
  • Heroku
  • PostgreSQL
  • oAuth
  • HTML5/CSS3/Javascript
  • MaterializeCSS

Explor

Travel App Built with Ruby on Rails

Explor is a travel app built in one week with two other web developer students. It is an API heavy application that fuels the wanderer inside us all. Explor was built with Ruby on Rails and with the intention to be easily added to for more robust functionality.

Technologies used

  • Ruby on Rails

  • Fullpage.JS
  • Heroku

  • SASS
